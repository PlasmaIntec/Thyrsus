## To Use
* `npm install`
* Add `config.js` with your [Google API Key](https://developers.google.com/maps/documentation/javascript/get-api-key)
* `npm run build`
* `npm start`
* Go to `localhost:3000`
* Enjoy!

## Example config.js
```javascript
// config.js
module.exports = {
	GOOGLE_API_KEY: 'YOUR_KEY_HERE'
}
```
